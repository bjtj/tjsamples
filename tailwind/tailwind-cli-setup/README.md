# Get started with Tailwind CSS #

<https://tailwindcss.com/docs/installation>


## How to test ##

1. Initialize

``` shell
npm i
```

or

``` shell
make init
```

2. Tailwind build & watch

Build only...

``` shell
make build
```

Or build and watch...

``` shell
make dev
```

3. Serve

``` shell
make serve
```

4. Open <http://locahost:3000/src>

# Getting Started #

<https://rocket.rs/v0.5-rc/guide/getting-started/>

```shell
rustup default stable
```

```shell
cargo new hello-rocket --bin
cd hello-rocket
```

Cargo.toml:

```toml
[dependencies]
rocket = "0.5.0-rc.2"
```

```shell
cargo run
```

# IPython & Jupyter #

## IPython ##

## Installing the classic Jupyter Notebook interface ##

<https://docs.jupyter.org/en/latest/install/notebook-classic.html>

Installation

``` shell
pip install jupyter
```

## Jupyter ##

Installation:

``` shell
pip install jupyterlab
```

Run:

``` shell
jupyter lab
```

Auto completion:

- Settings -> Settings Editor -> Code Completion -> Enable autocompletion.
